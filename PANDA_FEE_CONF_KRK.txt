CONFIGURATION FILE FOR PANDA FE
=============================================
Explanation: In this file consecutive settings for ASIC are saved. "settings for TCA" means settings for
asic number A (A can be of value 1 or 2), connected to tdc via cable connection number C (1-4) where
tdc is described by number T (1-...). After the line "settings for TCA" the settings go as following:
Amplification, Peaking time, TC1C2..0, TC1R2..0, TC2C2..0, TC2R2..0, For threshold and baseline given
number means exact value, whereas for the rest of the settings given number determines the position in
drop down list in web GUI.
